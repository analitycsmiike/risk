Hecho estilizado 1

Hecho estilizado 1: Los precios de los activos y el valor de los portafolios siguen una caminata aleatoria.

La siguiente grfica muestra el comportamiento de los precios:

plot(precio, main="Precio de las acciones de McDonald's")

Se puede observar el comportamiento no tiene una tendencia definida, ya que se experimentan periodos de alzas y bajas, por lo cual el comportamiento es aleatorio, no podemos definir con exactitud cual ser la tendencial en el siguiente punto del tiempo.

El comportamiento errtico de una variable aleatoria se describe por medio de un proceso estocstico que se llama camina aleatoria. Esto significa que el precio de un activo sigue el siguiente movimiento:

\[S_t=S_{t-1}+\epsilon\]

Donde \(E[\epsilon_t]=0\) y \(E[\epsilon_t, \epsilon_{t-1}]=Cov[\epsilon_t, \epsilon_{t-1}]=0\) Esto tiene las siguientes implicaciones: * No hay correlacin entre los retornos de un periodo con los anteriores. El rendimiento no tiene memoria. * Formacin eficiente de los precios, el precio actual de un activo refleja toda la informacin existente para darle valor, el ajuste de precios ante cambios se lleva a cabo rpidamente.

Anteriormente simulamos procesos estocticos. Vamos a generar un proceso de Wiener Generalizado para comparar el comportamiento de mi activo contra una simulacin:

set.seed(1234)

obs <- 2418
dt <- 1/obs
epsilon <- rnorm(obs, mean = 0, sd=1)

dW <- epsilon*sqrt(dt)

datos <- tibble(
  t=seq(from=dt, to=1, by=dt),
  dW
)

a <- 175
b <- 99

dx <- a*dt+b*dW

X <- 100

X[1] <- X

for(t in 2:obs){
  X[t] <- X[t-1]+dx[t]
}

datos <- mutate(datos, X=X, dX=dx)

cruce <- cbind(datos, activo)

cruce <- cruce %>% 
  select(precio, X, date) %>% 
  gather("Variable", "Valor", -date)

Vamos a graficar ambas variables para observar el comportamiento:

ggplot(cruce, aes(x=date, y=Valor, group=Variable))+
  geom_line(aes(color=Variable))

Hecho estilizado 3

Los rendimientos de los activos presentan volatilidad no constante y agrupada.

Analizamos las bandas de Bollinger:

chartSeries(to.monthly(precio), name="Precio de McDonald's", theme="white")

addBBands()

Las bandas que se muestran alrededor de la trayectoria del precio son las Bandas de Bollinger y miden la volatilidad del mercado. El 95% del precio deber situarse dentro de estas bandas, de forma resumida decimos que:

  • En un mercado ms voltil, las bandas de Bollinger se expanden
  • En un mercado menos voltil, las bandas se contraen

Hecho estilizado 4

La distribucin de los rendimientos presenta leptocurtosis.

La distribucin de los rendimientos es ms picuda y de colas pesadas. Esto implica que hay mayores posibilidades de obtener valores ms cercanos a la media, que en la distribucin normal.
